# Журналы ядра - это программа для записи логов ядра Linux в каталог /var/log.


# Лицензия
MIT 2006


# Функциональность
Функциональность данной программы:
* Записывает логи ядра текущей сессии;
* Записывает логи ядра всех сессий начиная с той, в которой программа начала работать.


# Установка
Шаг 1: Нужно скопировать файлы из репозитория
```bash
git clone https://www.github.com/SamSamych123/Kernel-logs
```

Шаг 2: Запустите файл install-kernel-logs.sh от имени root
```bash
sudo bash install-kernel-logs.sh
```

Шаг 3: Перезагрузитесь
```bash
sudo reboot now
```


# Просмотр логов
Для того, чтобы просмотреть логи текущей сессии:
```bash
sudo nano /var/log/kernel-logs.log
```

Для того, чтобы просмотреть логи всех сессий:
```bash
sudo nano /var/log/kernel-logs-all-sessions.log
```
